# Text Classification Project

This project aims to classify news articles into their respective categories using machine learning. The dataset used is the 20 Newsgroups dataset, which contains approximately 20,000 newsgroup documents, partitioned across 20 different newsgroups.

The project uses TF-IDF to transform the text data into a numerical representation, and then trains a Multinomial Naive Bayes classifier on the transformed data. The model is then evaluated using accuracy score and classification report.

The project also uses Grid Search to find the best parameters for the model.
