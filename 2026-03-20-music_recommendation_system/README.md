# Music Recommendation System

## Introduction
This project is a music recommendation system that uses natural language processing and collaborative filtering to suggest songs to users based on their listening history.

## Requirements
* Python 3.8+
* numpy
* pandas
* tensorflow
* scikit-learn

## Usage
1. Clone the repository
2. Install the required libraries by running `pip install -r requirements.txt`
3. Run the model by executing `python main.py`
4. Use the model to make predictions by creating a new dataframe with the song features and passing it to the `model.predict()` function
