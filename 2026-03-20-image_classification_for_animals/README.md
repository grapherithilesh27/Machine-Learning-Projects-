# Animal Kingdom Classifier

## Introduction
This project is a machine learning model that classifies images of different animals into their respective categories. The goal of this project is to train a model that can accurately classify images of animals.

## Dependencies
* tensorflow
* keras
* numpy
* matplotlib

## Usage
1. Install the required dependencies.
2. Download the dataset and replace 'path/to/train/directory' and 'path/to/validation/directory' with the actual paths to the dataset.
3. Run the main code to train the model.
4. Evaluate the model using the validation dataset.
5. Plot the training and validation accuracy to visualize the performance of the model.

## Example Use Cases
* Classifying images of animals in a zoo
* Identifying species of animals in a forest
* Classifying images of animals in a veterinary clinic
