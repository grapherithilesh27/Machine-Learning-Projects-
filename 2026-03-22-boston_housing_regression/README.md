# Boston Housing Price Prediction

## Introduction
This project uses the Boston housing dataset to predict the housing prices based on various features. The dataset is loaded from sklearn's datasets and split into training and testing sets. The model is trained on the scaled training data and evaluated on the scaled testing data.
## Dependencies
* sklearn
* numpy
* matplotlib
## Usage
1. Run the main.py file to train the model and make predictions.
2. The Mean Squared Error (MSE) is calculated and printed.
3. A scatter plot of the predicted vs actual values is displayed.
## Example Use Cases
* Real estate investors can use this model to forecast housing prices and make informed decisions.
* The model can be used to analyze the impact of various features on housing prices.
